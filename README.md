# GM_script
* 有道取词翻译.uc.js<BR/>
一个可以在浏览器中自由使用的屏幕取词脚本-alt+鼠标翻译，或者选中按Q翻译<BR/>
* AC-baidu： 优化百度、搜狗、谷歌搜索结果之重定向去除+去广告+favicon.uc.js<BR/>
去掉百度、搜狗、谷歌搜索结果的重定向问题；移除百度、搜狗部分广告；添加Favicon显示；添加图标计数功能<BR/>
* 搜索结果自动加序号.uc.js<BR/>
百度、搜狗、谷歌搜索结果添加序号，以及部分自定义网站<BR/>
